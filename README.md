# TicTacToe

TicTacToe web application.
Have a look-[TicTacToe](https://tictactoe.herokuapp.com/).

## Built With

* Node.js
* Express
* HTML
* Javascript
* CSS


## Installation:
1. git clone https://github.com/tejas002/tictactoe.git
2. cd tictactoe-master
3. npm install
```
$npm install
```
4. Run app
```
$npm start
```
